
# ❓ FAQ — Liviu Flow

---

### 1. Minhas tarefas não aparecem
Verifique:
- filtros ativos
- responsável configurado
- status selecionado

---

### 2. Como alterar o fluxo?
Admin → Configurações de Workflow

---

### 3. Posso vincular tarefas ao Liviu Content?
Sim, em **Vincular Conteúdo** dentro da tarefa.

---

### 4. As notificações não chegam
Verifique:
- permissões
- e-mail cadastrado
- se o navegador permite notificações

---

### 5. Como visualizar tarefas atrasadas?
Dashboard → Indicadores → “Atrasado”
