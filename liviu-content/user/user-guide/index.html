    📘 Guia Completo do Usuário — Liviu Content¶

    O Liviu Content é a ferramenta oficial da AdaptEdTech para criação, edição e publicação de conteúdos educacionais.
    Este guia completo irá ajudar você a dominar a plataforma, desde a criação de conteúdos básicos até gestão de colaboradores, revisão, publicação e integração com o LMS.

    1. Introdução ao Liviu Content¶

    O Liviu Content foi desenvolvido para que equipes educacionais e de T&D produzam materiais com qualidade, organização e eficiência, com foco em:

    • Criar conteúdos multimídia
    • Estruturar cursos, módulos e trilhas
    • Colaborar com times internos
    • Controlar versões e revisar materiais
    • Publicar no LMS com apenas um clique
    • Gerar dados analíticos automaticamente via SCORM/xAPI

    2. Perfis de Usuário¶

    Cada usuário pode ter permissões diferentes:

    Perfil Permissões
    Administrador Acessa tudo, gerencia pessoas e categorias
    Designer Instrucional Cria e edita conteúdos
    Redator Escreve e revisa textos
    Revisor Aprova, comenta, devolve revisões
    Gestor de T&D Acompanha fluxos, status e publicações

    3. Acesso à Plataforma¶

    1. Acesse a URL do Liviu Content
      https://content.adaptedtech.com (pode variar conforme cliente)

    2. Faça login pelo SSO AdaptEdTech.

    3. Você chegará à Dashboard Inicial, que exibe:

      • conteúdos recentes
      • rascunhos
      • itens aguardando revisão
      • conteúdos publicados recentemente

    4. Navegação Geral da Interface¶

    A interface é composta por:

    Menu Lateral¶

    • Criar Conteúdo
    • Meus Conteúdos
    • Em Revisão
    • Publicados
    • Templates
    • Configurações

    Topbar¶

    • Busca global
    • Nome do usuário
    • Acesso rápido ao SSO
    • Ajuda e suporte

    Área de Conteúdo¶

    Onde você visualiza:

    • Listas
    • Edição
    • Detalhes
    • Pré-visualização

    5. Criando um Conteúdo¶

    Para criar seu primeiro conteúdo:

    1. Clique em Criar Conteúdo
    2. Escolha o tipo:
      • Página
      • Módulo
      • Unidade
      • Curso
      • Quiz
      • Template
    3. Dê um nome inicial
    4. Clique em Criar

    Você será redirecionado ao editor.

    6. Usando o Editor do Liviu Content¶

    O editor é um sistema modular baseado em blocos.

    6.1 Estrutura do Editor¶

    Painel Central (conteúdo)¶

    Aqui você escreve e monta o material.

    Árvore de Seções¶

    Permite criar e ordenar:

    • módulos
    • unidades
    • aulas
    • sub-aulas
    • anexos

    Configurações Laterais¶

    Permite editar:

    • título
    • descrição
    • categoria
    • tags
    • objetivos de aprendizagem
    • capa
    • status

    7. Blocos Disponíveis¶

    O Liviu Content utiliza blocos inteligentes para montagem de conteúdo:

    Blocos de Texto¶

    • Título / subtítulo
    • Parágrafos
    • Citações
    • Destaques

    Blocos de Mídia¶

    • Imagens
    • Vídeos
    • PDFs
    • Áudios

    Blocos Interativos¶

    • Cards
    • Checklists
    • Listas
    • Quadros visuais

    Blocos Avaliativos¶

    • Questões de múltipla escolha
    • Verdadeiro ou falso
    • Dissertativas
    • Atividades práticas

    Blocos Técnicos (T&D corporativo)¶

    • Código
    • Tabelas
    • Caixas de informação

    Adicionar um bloco:

    1. Clique no símbolo +
    2. Escolha o bloco desejado
    3. Edite o conteúdo

    8. Organização do Conteúdo¶

    O Liviu Content permite organizar cursos completos.
    Você pode criar uma estrutura como:

    Curso → Módulo → Unidade → Aula → Seção → Blocos

    A organização é feita por:

    • arrastar e soltar
    • criar novas unidades
    • editar títulos
    • mover de posição

    9. Colaboração¶

    O Liviu Content oferece um fluxo colaborativo:

    Comentários Internos¶

    Comente em qualquer parte do conteúdo.

    Controle de Versões¶

    • Histórico completo
    • Comparação entre versões
    • Restauração de versões antigas

    Fluxo de Revisão¶

    Status disponíveis:

    • Rascunho
    • Em Revisão
    • Ajustes Necessários
    • Aprovado
    • Publicado
    • Revisores podem:
      • comentar
      • solicitar ajustes
      • aprovar
      • devolver

    10. Templates e Modelos¶

    O Liviu Content possui biblioteca de templates:

    • Aulas prontas
    • Introduções
    • Páginas de objetivo
    • Resumos
    • Quizzes padrão
    • Modelos corporativos

    Você pode:

    • usar um template existente
    • criar seus próprios modelos
    • duplicar conteúdos

    11. Publicando Conteúdos¶

    Para publicar:

    1. Clique em Publicar

    2. Escolha:

      • trilha
      • curso
      • categoria
      • grupos

    3. Confirme a versão

    4. Clique em Confirmar Publicação

    O conteúdo será:

    • enviado ao LMS
    • convertido em SCORM/xAPI (se aplicável)
    • disponibilizado aos alunos

    12. Exportações (opcional)¶

    Dependendo da configuração:

    • Exportar SCORM 1.2 / 2004
    • Exportar PDF
    • Exportar arquivo zipado

    Use o botão Exportar.

    13. Acompanhamento de Resultados¶

    Depois da publicação, os dados enviados via SCORM/xAPI podem ser vistos no:

    Liviu Analytics¶

    Você terá acesso a:

    • acessos
    • conclusão
    • desempenho em quizzes
    • tempo de estudo
    • indicadores por módulo/unidade

    14. Solução de Problemas (FAQ rápido)¶

    O conteúdo não salva¶

    • Verifique conexão
    • Veja se não há dois editores abertos
    • Recarregue a página

    Não consigo publicar¶

    • Verifique se o conteúdo está aprovado
    • Verifique permissões
    • Certifique-se de que o curso tem destino no LMS

    Imagem não aparece¶

    • Tamanho do arquivo
    • Permissão do servidor
    • Formato (prefira PNG/JPG)

    15. Suporte¶

    Se precisar de ajuda:

    • Consulte esta documentação
    • Use a central de ajuda AdaptEdTech
    • Abra um ticket no suporte
    • Consulte o administrador da sua instituição
